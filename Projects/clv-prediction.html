Data Science

Customer Lifetime Value Prediction

A machine learning pipeline built as my third capstone project for the Purwadhika Data Science certification, predicting Customer Lifetime Value (CLV) for a car insurance company using customer demographic and policy data. The project covers the full data science workflow — data cleaning, exploratory analysis, feature engineering, model benchmarking across 9 regression algorithms, and hyperparameter tuning to arrive at a final Gradient Boosting model.

Project Overview

Business Context: A car insurance company wanted to predict Customer Lifetime Value (CLV) using customer demographic and policy data, helping the marketing team target high-value customers more effectively and allocate budget more efficiently.
Data Cleaning: The data is cleaned using duplicate removal, missing value checks, and outlier detection across all numerical features.
Machine Learning: I use 9 regression models — including Linear Regression, KNN, Decision Tree, Random Forest, AdaBoost, XGBoost, and Gradient Boosting — evaluated using RMSE, MAE, and MAPE.
Hyperparameter Tuning: Gradient Boosting was selected as the final model and refined using RandomizedSearchCV and GridSearchCV hyperparameter tuning.
Machine Learning Prediction Result: Since policy count and premium amount are the dominant CLV drivers, the company can create more personalized offers based on these factors — supporting upselling, cross-selling, and more efficient retention of high-value customers.
